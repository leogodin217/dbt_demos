DBT Runtime Demo

* Learn the three phases of running DBT. 
* Understand why log messages appear in the console when they do.
* Use {% if execute %} to prevent DBT from running commands during inspection time.

### Using the starter project

Try running the following commands:
- dbt run
- dbt test
